## 更新方法

基本的には各ページの該当する文章や画像を差し替えれば更新できます。  
活動予定ページ（schedule.html）は、`<tr>`と`</tr>`で囲んでいる部分が行で`<td>`と`</td>`で囲んでいる部分が内容です。  
全体的にそんなに難しいことはしていないので少しググればたいていのことは解決すると思います。  
できるだけ毎年更新するようお願いします。
